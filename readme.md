## Dependencies

The released version of DPE-MEF was implemented and tested on Ubuntu 16.04 with:

- Python = 3.6.12
- PyTorch = 1.2.0
- torchvision = 0.4.0
- opencv-python = 4.4.0.46
- NVIDIA GPU + CUDA 10.0
